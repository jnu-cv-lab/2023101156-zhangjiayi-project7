# 手写数字识别实验说明（基于 sklearn digits）

## 一、实验目的

1. 掌握使用传统机器学习方法进行图像分类的基本流程；
2. 理解特征向量转换在图像分类中的必要性；
3. 对比不同分类器（KNN、朴素贝叶斯、逻辑回归、SVM、决策树、随机森林）在手写数字识别任务上的性能；
4. 学会分析错误样本及模型差异的原因。

## 二、实验环境

- 操作系统：Windows / Linux / macOS
- 编程语言：Python 3.x
- 主要库：scikit-learn、numpy、matplotlib、pandas

## 三、数据集信息

- 数据集名称：sklearn 自带的 digits 手写数字数据集
- 图像大小：8×8 像素（灰度，值范围 0～16）
- 图像数量：1797 张
- 类别：0～9 共 10 类
- 每张图像原始形状：(8,8)，展平后为 64 维特征向量

## 四、实验任务

### 任务1：数据准备
- 加载数据集，查看图像总数、单张图像尺寸、类别标签。
- 显示若干张样本图像及其真实标签。

### 任务2：数据划分
- 将数据集按 75% / 25% 划分为训练集和测试集。
- 说明训练集用于训练模型参数，测试集用于评估模型的泛化能力。

### 任务3：特征表示
- 将每张 8×8 图像转换为 64 维特征向量（reshape 或 flatten）。
- 解释为什么传统机器学习需要这种转换：模型要求固定长度的数值输入。
- 分析原始像素作为特征的优点（简单、直接）与局限（丢失空间结构，对平移/旋转/缩放敏感）。

### 任务4：模型训练
- 至少使用以下六种分类器：KNN、朴素贝叶斯（高斯）、逻辑回归、SVM（RBF核）、决策树、随机森林。
- 对每个模型：在训练集上训练，在测试集上预测，记录准确率。

### 任务5：结果比较
- 将所有模型的测试准确率整理成表格。
- 找出准确率最高和最低的模型。
- 分析不同模型表现差异的原因。

### 任务6：错误样本分析
- 选择表现最好的模型（本实验为 SVM）。
- 绘制混淆矩阵。
- 找出被误分类的样本，展示典型错误。
- 统计最容易混淆的数字对（如 8→1, 9→7, 5→9）。
- 分析错误原因：低分辨率下书写风格相似，像素特征难以区分。

## 五、实验结果汇总

### 测试准确率（按降序排列）

| 模型               | 准确率    |
|-------------------|----------|
| SVM (RBF)         | 99.11%   |
| KNN (k=3)         | 98.44%   |
| 逻辑回归           | 96.22%   |
| 随机森林           | 96.00%   |
| 朴素贝叶斯         | 82.89%   |
| 决策树             | 82.44%   |

### 最高与最低模型
- 最高：SVM（99.11%），仅错分 4 个测试样本。
- 最低：决策树（82.44%）。

### 错误分析（SVM）
- 总错误数：4 / 450。
- 错误对：
  - 8 → 1（2 次）
  - 9 → 7（1 次）
  - 5 → 9（1 次）
- 原因：低分辨率下部分数字书写风格接近（如 8 中间缝隙窄时类似潦草 1；5 与 9 的圆弧和竖线结构近似），像素特征无法捕捉细微形状差异。

## 六、实验结论

1. SVM 在本数据集上表现最佳，得益于最大间隔和核技巧。
2. 决策树和朴素贝叶斯较差，前者易过拟合，后者特征独立假设不成立。
3. 原始像素作为特征简单有效，但对空间变换敏感，且难以处理高度相似的手写风格。
4. 传统机器学习处理图像的基本流程为：加载 → 展平 → 划分 → 训练 → 评估 → 错误分析。
